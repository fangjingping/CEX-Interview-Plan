# 生产级落地清单（CEX）

## 目标
确保演示模块具备生产级系统的关键治理能力：一致性、可恢复性、安全、合规与可运维。

## 一致性与数据
- 事件日志：追加写、顺序保证、可回放。
- 快照机制：回放加速、校验点与校验工具。
- 幂等策略：请求 ID/事件 ID、去重表与 TTL。
- 对账体系：撮合/账本/钱包三方对账与修复流程。

## 高可用与容灾
- RTO/RPO 定义与演练记录。
- 多机房切换策略与数据回放机制。
- 降级策略：只读、限流、撮合暂停与风控兜底。

## 性能与容量
- 关键指标：P99 延迟、吞吐、消息积压。
- 压测基线：场景、容量与回归门槛。
- 热点治理：热点品种、热点 Key 与流量迁移。

## 安全与合规
- 密钥管理：HSM、轮换与权限隔离。
- 合规要求：KYC/AML、审计留存、监管报表。
- 风险防护：限流、反刷、异常检测与黑白名单。
- 数据留存：WORM 审计、归档策略与隐私合规。

## 运维与治理
- 可观测性：指标、日志、链路追踪与告警规则。
- 发布治理：灰度、回滚、配置中心与开关治理。
- 事故流程：故障定位、复盘、改进闭环。
- 服务治理：注册中心稳定性、限流/熔断/降级与重试策略。

## 交付检查（建议）
- 每个模块都有最小可运行 demo 与单元测试。
- 每个关键链路都有故障演练与恢复说明。
- 每个关键指标有监控与告警策略。
